Các code chỉ cần đổi đường dẫn từ trước folder NP_Project là có thể chạy được.
Cụ thể các file code như sau:
- Data_csv.ipynb: chuyển đổi dữ liệu gốc thành dữ liệu phù hợp cho bài toán ACD và APD
- ACD_Process: code chạy các mô hình cho bài toán ACD với trường hợp có tiền xử lý 
- ACD_Not_Process: code chạy các mô hình cho bài toán ACD với trường hợp không có tiền xử lý 
- APD_Process_head: code chạy các mô hình cho bài toán APD với trường hợp khía cạnh thêm vào đầu câu và có tiền xử lý
- APD_Process_tail: code chạy các mô hình cho bài toán APD với trường hợp khía cạnh thêm vào cuối câu và có tiền xử lý
- APD_Not_Process_head: code chạy các mô hình cho bài toán APD với trường hợp khía cạnh thêm vào đầu câu và không có tiền xử lý
- APD_Not_Process_tail: code chạy các mô hình cho bài toán APD với trường hợp khía cạnh thêm vào cuối câu và không có tiền xử lý
